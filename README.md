# Health-Tracking

## Demo

Try it out [here](https://elastic-volhard-eece4d.netlify.app/).

## Development

```bash
git clone git@github.com:HansKre/health-tracking.git
cd health-tracking
yarn install
yarn start
```

## Tech Stack

- `React` for user interfaces
- `craco` for `webpack`-config
- `styled-components` for styles
- `TypeScript`
- `netlify` for hosting
